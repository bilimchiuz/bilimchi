Политика конфиденциальности

Мы обрабатываем только те персональные данные, которые репетиторы добровольно предоставляют через Telegram-бот для публикации своих профилей в приложении Bilimchi.

Приложение не требует регистрации пользователей и не запрашивает персональные данные учеников. Профили репетиторов отображаются публично с их согласия.

Предоставленные данные используются исключительно для работы сервиса, отображения анкет репетиторов и улучшения качества платформы.

По вопросам конфиденциальности или для запроса удаления профиля вы можете связаться с нами по почте: bilimchisupport@yourdomain.com
